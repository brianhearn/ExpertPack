# Propositions

Atomic factual statements extracted from content files. Enables high-precision retrieval — when a user asks a specific factual question, RAG matches an exact proposition rather than a paragraph that happens to contain the answer.

Propositions are **derived** from content files — content files remain canonical. Update content files first, then regenerate propositions.
