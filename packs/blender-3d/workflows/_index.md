# Workflows

Step-by-step workflows for specific Blender tasks — practical guides that bridge the gap between knowing tools and knowing how to use them together.

- [Hard Surface Modeling](hard-surface-modeling.md) — The subdivision surface workflow, support loops, boolean cuts, bevel technique, modifier stacks, and troubleshooting shading artifacts for mechanical/manufactured objects.
- [Character Animation](character-animation.md) — Armature rigging (IK/FK, Rigify vs manual), weight painting, walk cycle construction, shape key lip sync, and the NLA workflow for mixing animation clips.
- [Product Visualization](product-visualization.md) — Studio lighting setup (three-point, HDRI hybrid), photorealistic PBR materials (metal, glass, plastic), turntable animation, and Cycles render settings for commercial product shots.
- [Motion Graphics](motion-graphics.md) — Geometry Nodes procedural animation, Scene Time-driven motion, shader-driven animation, camera animation with easing, and loop-ready animation setup.
- [Scene Optimization](scene-optimization.md) — Viewport and render performance, instances vs copies, adaptive subdivision, VRAM management, Cycles sampling strategy, and render farm preparation.
